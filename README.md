# Social Network Frontend (React + TypeScript + Vite)

This project is the frontend for your social network, built with React, TypeScript, and Vite.

## Prerequisites
- [Node.js](https://nodejs.org/) (version 16 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Installation

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <REPO_URL>
   cd "proyecto_red_social/frontend"
   ```

## UI Components Dependencies

### Core UI Components (Radix UI)
- `@radix-ui/react-avatar`: Avatar component
- `@radix-ui/react-context-menu`: Context menu functionality
- `@radix-ui/react-dialog`: Modal dialogs and alerts
- `@radix-ui/react-dropdown-menu`: Dropdown menus
- `@radix-ui/react-label`: Accessible form labels
- `@radix-ui/react-separator`: Visual separators
- `@radix-ui/react-tooltip`: Tooltips and popovers

### Animation and Effects
- `framer-motion`: Advanced animations and transitions
- `gsap`: Complex animations and timeline control
- `@react-three/fiber`: React renderer for Three.js
- `three.js`: 3D graphics and effects
- `@react-three/postprocessing`: Advanced visual effects

### Styling and Utilities
- `class-variance-authority`: Component variant management
- `clsx`: CSS class name utilities
- `tailwind-merge`: Tailwind class merging
- `tailwindcss`: Utility-first CSS framework
- `lucide-react`: Icon components

### Form Handling
- `react-hook-form`: Form state management
- `@hookform/resolvers`: Form validation resolvers
- `zod`: Schema validation

These dependencies are required for the components in `/src/components/ui/` directory.
